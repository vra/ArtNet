How It Works

ArtNet use Deep Learning to learn the style of certain artwork, then transfer your image to that style.
which is also known as style transfer

TensorFlow

This project is powered by the deep learning toolkit TensorFlow, which uses using data flow graphs for scalable machine learning.

fast-style-transfer

The back end of this project is based on fast-style-transfer. We use it to train model and run style transfer.

UIKit

The front end of this project is based on UIKit, which is a ightweight and modular front-end framework for developing fast and powerful web interfaces.

Alibaba Cloud

This project is deployed on Alibaba Cloud. A machine with 1GB memory, 1M bandwith is behind this project.

Features

ArtNet has several styles you can try, gives you cool results, with relatively fast speed.

Plentiful

We provide many styles you can transfer to. All of them are from famous artwork.

Cool

We use deep learning to learn the inner style of artworks. Create your awesome artwork by yourself.

Fast

Although we only use a CPU with only one core, we can finish your artwork in few seconds.
